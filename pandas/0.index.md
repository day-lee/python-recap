# Index

### 판다스에서 인덱싱은 데이터 선택해서 다루는 도구 
- 행 이름 Index
- 열 이름 Column

- 시리즈: 1차원 데이터 
- 데이터프레임: 2차원 데이터 

- df['컬럼명']: 브라켓 안에 `문자열` 전달, 해당 열 한개만 시리즈로 반환 
- df[['컬럼명']]: `리스트`를 전달, df로 반환. 
                리스트 아이템이 한개라도 한개 열을 가진 df로 나온다.  
                `df[['name', 'age', 'height']]`
- df[(조건식)]: `(조건식)`을 전달. 행을 필터링해서 df로 반환. 
              조건식은 ()괄호로 묶는다. 
              AND: () & () 
              OR: () | ()
              가독성을 위해 분리함. 
              kid_condition = (df['age'] < 12) | (df['height'] <= 150)
              kid_df = df[kid_condition]

## 🗺️ Pandas Indexing 전체 토픽 로드맵

### 1. 기본 인덱싱 (기반 다지기)
- **열(Column) 선택**: `df['A']` (시리즈) vs `df[['A']]` (데이터프레임)
- **행(Row) 슬라이싱**: `df[0:3]`처럼 숫자로 범위를 지정해 행 가져오기

### 2. 명시적 인덱서 (가장 중요한 핵심)
- **`df.loc[]` (Label-based)**: 컬럼명이나 인덱스 '이름'으로 행/열 선택하기
- **`df.iloc[]` (Integer-based)**: 0부터 시작하는 '정수 위치 번호'로 행/열 선택하기
- **`loc` vs `iloc` 차이점**: 슬라이싱할 때 끝값(상한선)이 포함되는지 여부 이해하기

### 3. 조건부 인덱싱 (실무 데이터 필터링)
- **불리언 인덱싱 (Boolean Indexing)**: `df[조건식]`을 활용해 원하는 행만 필터링하기
- **다중 조건 연결**: `&` (AND), `|` (OR), `~` (NOT) 연산자와 괄호`()` 활용법
- **`.isin()` 필터링**: 특정 리스트에 포함된 값들만 한 번에 쏙 골라내기 (예: `df['도시'].isin(['서울', '부산'])`)

### 4. 인덱스 제어 및 조작 (표의 축 바꾸기)
- **`.set_index()`**: 특정 컬럼을 표의 '행 인덱스(이름)'로 지정하기
- **`.reset_index()`**: 기존 인덱스를 일반 컬럼으로 되돌리고 0, 1, 2... 순정 인덱스로 초기화하기
- **인덱스 이름 변경**: `.rename(index=...)` 또는 `.rename(columns=...)` 활용법

### 5. 고급 인덱싱 (이후에 배우게 될 심화 과정)
- **멀티 인덱스 (MultiIndex / Hierarchical Indexing)**: 행이나 열에 계층(여러 층)을 만들어 복잡한 데이터 다루기
- **`.loc`을 이용한 데이터 수정**: 특정 조건의 데이터만 안전하게 값을 덮어쓰기 (SettingWithCopyWarning 방지)
