## 🎯 LeetCode 30 Days of Pandas 전체 토픽 리스트

### 1. Data Filtering (데이터 필터링 - 현재 단계!)
- **불리언 인덱싱**: 특정 조건에 맞는 데이터만 추출 (`df[조건식]`)
- **다중 조건 연산**: `&` (AND), `|` (OR) 연산자와 괄호 사용법
- **Null 값 처리**: `isna()`, `notna()`를 활용한 데이터 유무 필터링

### 2. String Methods (문자열 처리)
- **문자열 변환**: `.str.upper()`, `.str.lower()`, `.str.capitalize()` (첫 글자만 대문자로)
- **문자열 조건 검색**: `.str.contains()`, `.str.startswith()`
- **정규표현식(Regex) 활용**: 특정 이메일이나 패턴을 만족하는 텍스트 필터링 (`.str.match()`)

### 3. Data Manipulation (데이터 변경 및 정렬)
- **값 정렬**: `.sort_values()` (오름차순, 내림차순 및 다중 컬럼 정렬)
- **중복 제거**: `.drop_duplicates()` (중복된 데이터 제거하기)
- **값 수정 및 파생 변수**: 조건에 따라 새로운 컬럼 만들기 (`np.where` 또는 `.apply()`)
- **피벗 테이블**: 행과 열의 구조를 바꾸기 (`.pivot()`, `.melt()`)

### 4. Data Aggregation & Grouping (그룹화 및 통계)
- **그룹화**: `.groupby()` (특정 컬럼 기준으로 데이터 묶기, SQL의 GROUP BY)
- **통계치 계산**: `.agg()`, `.sum()`, `.count()`, `.mean()`
- **고유값 개수 세기**: `.nunique()` (중복을 제외한 유일한 데이터 개수, SQL의 DISTINCT COUNT)
- **순위 매기기**: `.rank()` (가장 높은 급여나 점수 순으로 순위 부여하기)

### 5. Data Integration (데이터 병합 및 조인)
- **데이터프레임 합치기**: `.merge()` (두 개 이상의 표를 결합하기, SQL의 JOIN)
- **내부/외부 조인 옵션**: `how='inner'`, `how='left'`, `how='right'`, `how='outer'` 이해하기

### 6. Advanced Statistics (심화 데이터 처리)
- **조건별 카테고리화**: 복잡한 비즈니스 로직에 맞춰 데이터를 상/중/하 등으로 분류하고 통계 내기
- **시계열/날짜 데이터 처리**: 날짜형 데이터 비교 및 연산
