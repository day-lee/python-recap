### 날짜 범위 필터링
- BETWEEN 방식 (가장 추천하는 직관적인 방식, 두 날짜를 모두 포함함)
WHERE order_date BETWEEN '2020-02-01' AND '2020-02-29' 

- AND 방식 
WHERE '2020-02-01' <= order_date AND order_date <= '2020-02-29' 
컴퓨터는 한번에 세개의 값을 비교하지 못한다. e.g. '2020-02-01' <= order_date <= '2020-02-29' 에러!

# POSTgresql 

### 날짜 추출 
- YEAR FROM, MONTH FROM, DAY FROM, HOUR FROM, MINUTE FROM
- 'year, 'month', 'day', 'hour', 'minute'
EXTRACT(YEAR FROM date) as extracted_year 
DATE_PART('year', date) as part_year  

### 날짜 단위 round down 
- DATE_TRUNC()
- DATE_TRUNC('month', date) -> 08/01/2022 00:00:00 (8월이면 08/01로 맞춰버림)

### 인터벌 추가, 제거 
- date + INTERVAL '2 days' 
- date - INTERVAL '10 minutes' 

### date, timestamp를 특정 포맷으로 변경 
- TO_CHAR(date, 'YYYY-MM-DD HH24:MI:SS')
- ISO 8601 Date and Time	'YYYY-MM-DD HH24:MI:SS'

### 타입변경: 스트링을 date, timestamp로 변경
- string -> date:   ::DATE or TO_DATE('2026-08-29', 'YYYY-MM-DD') 
- string -> timestamp:   ::TIMESTAMP, TO_TIMESTAMP('2026-08-29 10:30:00', 'YYYY-MM-DD HH:MI:SS')
